Custom User-Agent

Simple patch to set a custom User-Agent. Add to the "Choices" the following strings

useragent : myUserAgent

If useragent is null or not set, the default User-Agent of Netsurf will be used

Advantage of a custom one

It add somehow, a bit of privacy by not giving away what browser you are using

Many website doesn't not render properly or are unusable, example with Google

Display of Google using custom User-Agent (Lynx/2.9.0dev.5 libwww-FM/2.14 SSL-MM/1.4.1 OpenSSL/1.1.1g)

With the custom User-Agent, the menu is properly displayed, we can switch to Images / News / Videos...

Patch for netsurf 3.10
