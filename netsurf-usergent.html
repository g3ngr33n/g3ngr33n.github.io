Overwrite the useragent

Edit .config/netsurf/Choices / usergent: value

if useragent not set or set to null, default User-Agent is used (NetSurf/3.10 (Linux))


Using a custom User-Agent allow many website to work better, example with Google



Display of Google using default useragent : NetSurf/3.10 (Linux)



Display of Google using custom useragent : Lynx/2.9.0dev.5 libwww-FM/2.14 SSL-MM/1.4.1 OpenSSL/1.1.1g

Patch for netsurf 3.10
